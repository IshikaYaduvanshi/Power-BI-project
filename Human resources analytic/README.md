# Human Resources Analytics Dashboard

## 📌 Objective

The objective of this project is to analyze employee data and workforce performance using an interactive Power BI dashboard. This dashboard helps organizations monitor employee satisfaction, attrition trends, recruitment performance, and hiring patterns to support better HR decision-making.


## 📊 Dashboard Features

* Employee Attrition Analysis
* Recruitment Source Tracking
* Department-wise Performance Analysis
* Employee Satisfaction Monitoring
* Hiring Trend Visualization
* Interactive Filters and Slicers
* Location-based Attrition Insights


## 📈 Key KPIs

* Average Satisfaction Score
* Total Employees
* Average Performance Rating
* Attrition Rate

## 📍 Insights Generated

* Attrition rate is high (~50%), indicating employee retention issues.
* Employee satisfaction and performance are average across departments.
* Hiring demand is increasing to offset employee exits.
* Attrition is consistent across multiple locations.
* Recruitment sources contribute almost equally to hiring.


## 🛠 Tools & Technologies Used

* Power BI
* Microsoft Excel
* DAX
* Data Visualization Techniques
