# Financial Health Dashboard

## Objective

Developed an interactive Power BI dashboard to analyze an organization's financial performance and provide real-time insights for SMEs.

## Features

* Income statement analysis
* Balance sheet visualization
* Cash flow monitoring
* Profitability trend analysis
* Financial forecasting
* Interactive filters and slicers

## Tools Used

* Power BI
* Excel
* DAX

## Dashboard Insights

* Revenue and expense comparison
* Net profit analysis
* Regional profitability tracking
* Cash balance monitoring
* Asset vs liability analysis
* Forecasting for budgeting and planning
